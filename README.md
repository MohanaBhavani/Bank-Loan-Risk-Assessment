🏦 Bank Loan Risk Assessment

A machine learning project developed to assess loan applications and predict the risk of loan defaults using customer and financial data. The system helps financial institutions make faster and more accurate lending decisions.

✨ Technologies

- Python
- SQL
- Scikit-learn
- XGBoost
- Flask

🚀 Features

- Predicts loan approval and default risk
- Data preprocessing and feature engineering
- Machine learning-based risk analysis
- Real-time prediction support using APIs
- Model evaluation with business-focused metrics

📌 Project Workflow

The workflow starts with collecting customer financial and loan-related datasets. Data preprocessing is performed to handle missing values, categorical variables, and feature selection. Machine learning models such as Logistic Regression and XGBoost are trained to classify high-risk and low-risk applications. The best-performing model is deployed through Flask APIs for real-time loan risk prediction.

🛠 How to Run

1. Clone the repository  
2. Install dependencies using `pip install -r requirements.txt`  
3. Start the Flask server  
4. Send loan application data through the API  
5. View prediction and risk assessment results
